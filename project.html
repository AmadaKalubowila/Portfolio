Sheild Life Insurance

Sheild Life Insurance is an automated system for an insurance company. To complete this system, we mainly used HTML, CSS, PHP as languages, and Xampp as a database provider. This system mainly includes 5 CRUD functions: user registration, review and rating, online agent, online service, and admin privacy policy. Inventing this system, we want to give a user-friendly, and easy system to customers.

KIU Smart Cafeteria

KIU Smart Cafeteria is an automated lunch ordering system. To complete this system, we are mainly using Java, HTML, CSS, JavaScript, MySQL as languages. Also, we are using Spring Boot as the backend framework and React as the frontend framework, with MySQL as the database provider too. This system has many functions like user registration, inventory, ordering, billing, review, vendor registering, and security view too. Inventing this system, we want to give an easy method to handle orders in the cafeteria.

Gentle Vaccine

Gentle Vaccine is an automated vaccine system. To complete this system, we are mainly using Java, HTML, CSS, JavaScript, MySQL as languages. Also, we are using Spring Boot as the backend framework and React as the frontend framework, with MySQL as the database provider too. This system has many functions like user registration, auto-scanning, notifying, and more. Inventing this system, we want to give an easy method to handle vaccinations in hospitals and keep track of them correctly.

FreshSip

FreshSip is an automated juice bar system. To complete this system, we are mainly using Java, HTML, CSS, JavaScript, MySQL as languages. Also, we are using Spring Boot as the backend framework and React as the frontend framework, with MySQL as the database provider too. This system has many functions like user registration, ordering, billing, reviews, and carts. Inventing this system, we want to give an easy method to handle orders in the juice bar and provide a place that satisfies all the needs of the user.
